# Genie Logiciel – Cinema BI

Application web développée en **Nuxt** permettant l’analyse et la visualisation de données cinéma
à partir de l’API TMDb (filtres, indicateurs, dashboard).

## Stack technique
- Nuxt 4 (front + server routes)
- Tailwind CSS
- MySQL
- Docker (environnement de développement)
- GitHub Actions (CI/CD)

## Lancer le projet en développement (local)
```bash
npm install
npm run dev
